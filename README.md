# Smart Platform - UI
UI for Smart Platform
### Build
```
$ docker build -t smart-ui:1.0.0 .
```
### Start
```
$ docker run -d -p 8001:8001 smart-ui:1.0.0
```
### Tools
- Public URLs for sending previews: https://ngrok.com/
